# Αιγαιομεταφορική — Landing Page Design Spec

**Date:** 2026-03-15
**Client:** Αιγαιομεταφορική Ε.Π.Ε. — Αφοί Κατσούρη
**Goal:** Marketing landing page για απόκτηση νέων πελατών (lead generation)

---

## 1. Πλαίσιο

Η Αιγαιομεταφορική Ε.Π.Ε. είναι εταιρεία **B2B εμπορευματικών μεταφορών** με παρουσία από το 1990. Εξυπηρετεί επιχειρήσεις με μεγάλα φορτία — δεν είναι courier και δεν κάνει μικροδέματα. Καλύπτει Αττική, πανελλαδικά δρομολόγια, νησιά Αιγαίου (Κάλυμνος, Κως, Ρόδος) και μετακομίσεις. Χρειάζεται landing page που να προσελκύει επαγγελματίες πελάτες και να τους ωθεί να επικοινωνήσουν.

---

## 2. Τεχνολογία

- **Styling:** Tailwind CSS
- **Deploy:** Vercel

## 5. Λειτουργικές Απαιτήσεις

- **Email:** Resend API, Server Action (όχι API route)
- **Responsive:** **Mobile-first — κύρια προτεραιότητα.** Η σελίδα πρέπει να είναι εξαιρετική στο κινητό, όχι απλώς "λειτουργική". Breakpoints: sm(640) md(768) lg(1024) xl(1280)
- **Mobile UX specifics:**
  - Top bar κρύβεται σε mobile, nav γίνεται hamburger drawer
  - Hero: single column, φόρμα κάτω από content, μεγάλα touch targets (min 48px)
  - CTA τηλεφώνου: `<a href="tel:...">` για άμεσο dial
  - Κάρτες services: single column stack
  - Φόρμα: μεγάλα inputs, εύκολη πληκτρολόγηση σε κινητό
  - Footer: single column
  - Sticky "📞 Καλέστε μας" bar στο κάτω μέρος της οθόνης (mobile only) για άμεση επικοινωνία
- **SEO:** title, meta description, OG tags, favicon
- **Performance:** SSG, next/image για φωτογραφίες
